Glossary

Glossary#

Autenticação de 2 fatores#

2FA é um meio para um usuário de computador se identificar e autenticar usando tanto algo que ele sabe (geralmente uma senha) quanto algo que ele possui (frequentemente um telefone ou chaveiro com dígitos que mudam periodicamente).

Bit#

Um único um ou zero; o dígito fundamental no sistema de numeração binário.

Byte#

Oito bits juntos, representando um número entre 0 e \(2^8-1\) ou 255

Bitmap#

Gráficos de computador representados como uma grade conectada de pixels de diferentes cores que formam uma imagem. Usados frequentemente para representar fotografias em computadores.

Folhas de Estilo em Cascata - CSS (Cascading Style Sheets)#

Um sistema de arquivos de texto que definem a aparência e o comportamento de uma página web separadamente dos arquivos e/ou dados que compõem o conteúdo da página.

CMS#

Um sistema de gerenciamento de conteúdo é um software baseado na Web que auxilia na construção e manutenção de sites de forma amigável ao usuário.

Nuvem#

Servidores de computador sempre disponíveis executando na internet. Frequentemente usados como um local alternativo para armazenar arquivos ou e-mails em vez de em um disco rígido local.

Linha de comando#

A interface em muitos sistemas de computador que permite ao usuário inserir comandos de texto para serem processados. Infame por parecer complicada, mas na verdade sendo extremamente expressiva e útil.

Unidade Central de Processamento#

A CPU é o "cérebro" de um computador, é um microprocessador que processa todos os números. Feita de bilhões e bilhões de pequenos transistores gravados em uma pastilha de silício.

Dark web#

Qualquer parte da internet que não está prontamente disponível para navegação pelo público. Tecnicamente inclui os e-mails dos seus amigos, já que nem todos estão autorizados a vê-los. Tipicamente usado na prática para se referir aos mercados negros que funcionam junto com serviços mais legítimos no Roteador Onion.

Banco de dados#

Um sistema de computador projetado para receber, armazenar e recuperar grandes quantidades de informações de múltiplos outros sistemas de computador simultâneos.

DNS#

O Sistema de Nomes de Domínio é uma "lista telefônica" para redes de computadores que mapeia nomes legíveis como www.webpage.com para endereços IP utilizáveis por computadores como 255.255.255.20.

EFF#

Electronic Frontier Foundation é uma organização sem fins lucrativos que defende privacidade digital, liberdade de expressão e inovação.

Criptografia#

O processo de ocultar informações de olhares indiscretos em rede ou em outros lugares usando matemática. Também pode validar que a informação foi enviada sem alterações de um indivíduo específico.

GNU#

Gnu's Not Unix (GNU) (acrônimo recursivo) é um projeto de colaboração de software livre que começou em 1983 e continua até hoje. Muitas ferramentas livres e de código aberto são desenvolvidas sob o guarda-chuva do GNU.

GUI#

Uma Interface Gráfica do Usuário usa um monitor e um mouse ou tela sensível ao toque para fornecer uma experiência baseada em visual no uso de um programa de computador em vez de um ambiente puramente baseado em texto. Windows e macOS são sistemas operacionais com GUI, enquanto o DOS não era (se você se lembra do DOS).

Disco rígido#

O dispositivo em um computador que armazena informações como documentos, fotos, programas, e assim por diante por longos períodos de tempo. Contrasta com RAM (veja Memória).

Endereço IP#

Um endereço de Protocolo de Internet define onde um computador está em uma rede de computadores, semelhante a um endereço de rua na vida real. É numérico, seja quatro números entre 0 e 255 separados por pontos (IPv4) ou oito grupos de quatro dígitos hexadecimais separados por dois pontos (IPv6). Todos os nomes de domínio e outros locais de rede legíveis por humanos são mapeados para endereços IP via DNS.

ISP#

Um Provedor de Serviços de Internet é uma empresa que você paga para fornecer uma conexão à internet pública. Exemplos incluem Claro, Vivo, Oi, etc.

LaTeX#

Um sistema de composição tipográfica e publicação de documentos que separa conteúdo da apresentação.

Linguagem de marcação leve - Light Markup#

Uma linguagem de marcação com uma sintaxe intencionalmente simples, frequentemente para acomodar a criação rápida por um escritor humano.

Regressão linear#

O processo de encontrar a equação de uma linha que melhor se ajusta a um determinado conjunto de pontos de dados.

Linux#

Um sistema operacional de código aberto que alimenta muitos servidores que compõem a internet, todos os smartphones Android, todos os 500 maiores supercomputadores do planeta e um número pequeno, mas crescente, de PCs.

macOS#

Um sistema operacional proprietário feito pela Apple e usado em computadores Apple e iPhones.

Linguagem de marcação - Markup#

Uma maneira de expressar algo para um computador usando um arquivo de texto com anotações especiais baseadas em texto. Por exemplo, enquanto o itálico não pode ser representado diretamente em um arquivo de texto puro, uma palavra poderia ser indicada como em itálico cercando-a com tags <i>. Linguagens de marcação são frequentemente usadas para troca de dados ricos (por exemplo, HTML e XML).

Memória#

Hardware que pode armazenar grandes pedaços de informação em um computador por algum período de tempo. Memória é tipicamente usada para Memória de Acesso Aleatório (RAM), que é uma memória de curta duração e ação rápida. Há também a memória do disco rígido, que é mais lenta, maior e mais barata.

Armazenamento Conectado à Rede#

Um NAS é um banco de grandes discos rígidos que são disponibilizados em uma rede, frequentemente usados para backups de fotos e vídeos.

Ruído#

Informação aleatória presente em qualquer sinal analógico, como o chiado no rádio ou a "neve" nas TVs antigas

Roteador Onion#

Uma tecnologia de navegação na web que oferece maior anonimato e segurança em comparação com a navegação normal. Ela esconde as páginas da web que você está navegando da infraestrutura e esconde sua origem das páginas da web que você navega.

Sistema operacional#

Um SO é um conjunto de software que executa um computador. Ele interage com todo o hardware e fornece o ambiente no qual outros programas são executados. Exemplos são Windows, macOS e Linux.

Gerenciador de pacotes#

Um programa que pode acessar a internet e encontrar um programa diferente, baixá-lo, instalá-lo, configurá-lo e (mais tarde, se desejado) desinstalá-lo. Estes precedem as lojas de aplicativos, mas são muito semelhantes em conceito. Eles permitem um fácil gerenciamento de programas, frequentemente através de canais pelo menos nominalmente mais seguros do que downloads aleatórios. Cada sistema operacional tem seus próprios gerenciadores de pacotes.

Gerenciador de senhas#

Um programa ou serviço que armazena várias senhas para vários serviços em um único lugar. Também possui recursos para gerar senhas aleatórias para evitar os problemas de reutilização de senhas.

Programação#

Escrever uma sequência de operações para um computador executar. O processo de criar software.

Expressões regulares - REGEX#

Um padrão geral representando alguma combinação de letras, números, pontuação, e outros caracteres que é usado para pesquisa avançada e substituição em texto e processamento de dados.

Roteador#

Um componente de rede que recebe pacotes e os encaminha para um cliente ou outra rede que ele conhece, mas talvez o remetente não conheça (e vice-versa).

Shell Seguro (SSH)#

Um meio para acessar a linha de comando de um computador que está conectado a você por uma rede que mantém forte segurança ao longo da rede enquanto comandos e respostas são passados de um lado para o outro.

Gerador de site estático#

Um programa ou sistema que converte código de linguagem de marcação leve e imagens em um site HTML completo. É efetivamente um CMS leve.

Editor de texto#

Um programa que visualiza e modifica arquivos de texto.

Arquivo de texto#

Um arquivo universal que contém apenas sequências de caracteres sem qualquer informação incorporada sobre fonte, estilo, etc. Estes são muito úteis e fundamentais como um arquivo de interface entre várias etapas de processamento de dados, publicação, programação, etc.

Domínio de primeiro nível#

Um TLD é o sufixo de um nome de domínio como .com ou .edu ou mesmo .horse.

Gráficos vetoriais#

Gráficos de computador definidos por equações matemáticas em vez de grades de pixels. Para certos tipos de imagens (por exemplo, ícones, logotipos), eles são bons porque usam muito poucos bits, mas podem ser ampliados infinitamente. Contrasta com Bitmaps.

Máquina virtual#

Um programa de computador que emula o hardware de um sistema de computador diferente.

VPN#

Uma Rede Virtual Privada é uma tecnologia que conecta com segurança através da internet pública a servidores nos quais você está especialmente autorizado. Usada frequentemente para empresas, permitindo que funcionários se conectem a recursos de intranet corporativa como unidades compartilhadas enquanto viajam. Também oferecida como um serviço comercial para ajudar a adicionar segurança e privacidade de rede para consumidores.

VST#

Virtual Studio Technology é uma especificação que define como um grande conjunto de plugins de software trabalham juntos ao compor música em um estúdio computadorizado.

Windows#

Um sistema operacional proprietário feito pela Microsoft que roda na maioria dos PCs de consumidores e escritórios.
